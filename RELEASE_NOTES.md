# FreeSafeAuth Personal v1.0.1

## English

UI refresh and biometric unlock improvement.

No ads, no analytics, no login, no server. FreeSafeAuth Personal remains a local-only TOTP authenticator for personal use.

- Refreshed the app UI with a green security theme matching the app icon.
- Improved the home screen, token cards, settings page, add-account flow, and lock screen.
- Face unlock now requests immediate entry after successful authentication without an extra confirmation tap.
- Includes the previous v1.0.0 local TOTP, encrypted backup, import preview, duplicate detection, biometric unlock, screenshot protection, and multi-language support features.
