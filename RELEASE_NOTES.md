# FreeSafeAuth Personal v1.0.0

Initial MVP release.

## Highlights

- Local TOTP authenticator for personal use
- Manual and QR-code account setup
- Encrypted local secret storage with Android Keystore + AES-GCM
- Biometric unlock and auto-lock options
- Encrypted backup export/import
- Optional encrypted backup email handoff via system mail app
- Chinese, English, Japanese, and Korean UI support
- No ads, no analytics, no account login, no server
- No network permissions

## APK

This release contains a debug APK for testing. For production installation and distribution, build a signed release APK/AAB.
